🎮 2048 Game (Full Stack Project)

A modern full-stack implementation of the classic 2048 puzzle game, built using HTML, CSS, JavaScript (frontend) and Node.js, Express, MongoDB (backend).

This project allows users to play the game and automatically save/load their progress from a database.

🚀 Features
🎮 Classic 2048 gameplay (4x4 grid)
⌨️ Keyboard controls (Arrow keys)
💾 Save game progress to database
📥 Load previous game session
📊 Score tracking
🌐 Full-stack architecture
🗄️ MongoDB storage per user
⚡ Fast and lightweight API

🎮 How to Play
Use Arrow Keys to move tiles
Combine same numbers to increase score
Game auto-saves after every move
Refresh anytime and continue progress
